To get the Marchosias Follower Form you will need to download [BepinEx](https://github.com/BepInEx/BepInEx/releases), [COTL Custom Spine Loader](https://github.com/InfernoDragon0/COTL-CustomSpineLoader), [COTL API](https://github.com/xhayper/COTL_API) and [CotlSaveExtractorLoader](https://github.com/osoclos/CotlSaveExtractorLoader).

Then you need to but the [BepinEx](https://github.com/BepInEx/BepInEx/releases) folder into Cult Of The Lamb's files (you can find them by selecting "Browse Local Files" on Steam).

Then you need to start and then close the game.

After that you need to put the [COTL Custom Spine Loader](https://github.com/InfernoDragon0/COTL-CustomSpineLoader), [COTL API](https://github.com/xhayper/COTL_API) and [CotlSaveExtractorLoader](https://github.com/osoclos/CotlSaveExtractorLoader) into the plugins folder inside [BepinEx](https://github.com/BepInEx/BepInEx/releases) (if it's not there you can just create a folder named "plugins" and it'll work).

Then you need to put the "Marchosias" folder inside the "FollowerSkins folder in [COTL Custom Spine Loader](https://github.com/InfernoDragon0/COTL-CustomSpineLoader) (if it's not there you can just create a folder named "FollowerSkins" and it'll work).

Lastly you need to open the game, go into your save file, save the game and go back to the main menu.

From there you'll get a "-extracted.json" file in your saves folder(in AppData), you need to put that file into a [JSON Editor](https://jsoneditoronline.org/)(i like using this one) and go to "FollowerSkinsUnlocked" and add "Marchosias_base" to the end.

If you did all of the steps right he should appear in the follower forms sections in game.

Make sure to have fun with your very own Marchosias.
